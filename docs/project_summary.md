# Project Summary

This project implements a Python-based EEG analysis pipeline for resting-state EEG and random-dot kinematogram (RDK) visual motion task EEG.

The workflow includes preprocessing, quality control, spectral feature extraction, aperiodic EEG parameterization, task-feature extraction, rest-task feature merging, exploratory correlation analysis, and a basic machine-learning demonstration.

The final matched rest-task sample included six participants after quality-control exclusions. Therefore, the statistical and machine-learning outputs are interpreted as exploratory and primarily demonstrate reproducible EEG workflow implementation.
